# GPSend - GriefPrevention Addon!

A simple addon that allows players to transfer claim blocks to each other! <br>
Feel free to contribute as I currently don't have time to maintain.

Download releases: https://github.com/BrihtaKai/GPSend/releases  <br>
Download dev builds: https://github.com/BrihtaKai/GPSend/actions

## Requirements

- [GriefPrevention](https://modrinth.com/plugin/griefprevention) (**required**)
- [PlaceholderAPI](https://modrinth.com/plugin/placeholderapi) (optional, to use placeholders in GUI/messages etc.)

## Commands

| Command | Description | Permission |
|---------|-------------|-----------|
| `/gpsend` | Open claim block transfer GUI | `gpsend.send` |
| `/gpsend player <player> <amount>` | Send blocks to specific player | `gpsend.send` |
| `/gpsend all` | Send blocks to all online players | `gpsend.sendall` |
| `/gpsend reload` | Reload configuration | `gpsend.admin` |
